# InfoIcon

The InfoIcon component creates an generic information icon using inline SVG.
This icon is typically used to for tooltips or help indicators.


```javascript

```

### Changing the Color

You can adjust the color of the icon using the `iconFill` or the `circleFill` prop. These props accept common color names and hex values.
