# ArrowLineIcon

The ArrowLineIcon component creates an arrow with a line icon using inline SVG.

### Changing the Color

You can adjust the color of the icon using the `fill` prop. This prop accepts common color names and hex values.

<Preview>
  <ArrowLineIcon height="3rem" width="3rem" fill="#ababab" />
</Preview>

```javascript

```

Using the `direction` prop you can adjust the direction that the arrow points.

</Preview>

