# HouseIcon

The HouseIcon component creates a house icon (or multiple house icons) using inline SVG.

### Animating the Houses

You can animate the fill on the houses by providing a `highlight` prop to the component. If you want to display `100` houses, and highlight `50` of them you do so. The fill animation is disabled for Internet Explorer users.

<Preview>
  <HouseIcon height="3rem" width="3rem" houses={100} highlight={50} />
</Preview>

```javascript
<HouseIcon height="3rem" width="3rem" houses={100} highlight={50} />
```
