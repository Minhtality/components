# CompassCircleIllustration

The CompassCircleIllustration component creates a compass illustration using inline SVG. This illustration is commonly used within McGonnagal introduction cards and on United Income 404 page.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import CompassCircleIllustration from '@unitedincome/components/dist/CompassCircleIllustration';
```

```javascript
import {CompassCircleIllustration} from '@unitedincome/components';
```

### Animation

The compass illustration can be animated using a combination of the `isAnimated` and `animationDuration` props. The duration prop is in seconds and can accept decimals.

<Preview>
  <CompassCircleIllustration
    height="14rem"
    width="14rem"
    isAnimated={true}
    animationDuration={0.3}
  />
</Preview>

```javascript
<CompassCircleIllustration
  height="14rem"
  width="14rem"
  isAnimated={true}
  animationDuration={0.3}
/>
```
