# ShoppingCircleIllustration

The ShoppingCircleIllustration component creates a shopping bag illustration using inline SVG. This illustration is commonly used within McGonnagal introduction cards.

## Importing ðŸ“¦

You can use this component using one of the following import patterns.

```javascript
import ShoppingCircleIllustration from '@unitedincome/components/dist/ShoppingCircleIllustration';
```

```javascript
import {ShoppingCircleIllustration} from '@unitedincome/components';
```

### Draft

You can use the `draft` prop to render the illustrations draft mode variant.

<Preview>
  <ShoppingCircleIllustration height="14rem" width="14rem" draft />
</Preview>

```javascript
<ShoppingCircleIllustration height="14rem" width="14rem" draft />
```

### Color

You can use the `color` prop to render the illustrations full color variant.

<Preview>
  <ShoppingCircleIllustration height="14rem" width="14rem" color />
</Preview>

```javascript
<ShoppingCircleIllustration height="14rem" width="14rem" color />
```

### Illuminate

The `illuminate` prop activates the interaction variant of the illustration. By default the interaction occurs when the illustration is clicked, this prop allows you to override that behavior so you can trigger it as a side effect of another action.

<Preview>
  <ShoppingCircleIllustration height="14rem" width="14rem" illuminate />
</Preview>

```javascript
<ShoppingCircleIllustration height="14rem" width="14rem" illuminate />
```
